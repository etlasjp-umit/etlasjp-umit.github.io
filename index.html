OEM/ODM 能力

  • 最小起订量参考：3,000 单位起（依品类协商）
  • 配方定制：可选鱼胶原/植物替代、香精/无香、敏感肌版本
  • 包装定制：金箔烫印、蓝/粉/紫主题、可挂孔单片/精品礼盒
  • 认证支持：Halal、GMP、ISO；可配合准备 COA、MSDS、自由销售证明等出口资料

开发流程（示例）

  1. 需求沟通 → 目标市场/法规确认
  2. 配方与包装提案 → 样品打样
  3. 稳定性/微生物等测试 → 合规审核
  4. 下单生产 → 日本灌装/复检 → 出厂

* 可根据沙特、阿联酋、土耳其等地法规及清真标准进行本地化适配，交期视配方复杂度与认证需求而定。

合规与认证

Halal

支持 NAHA / BPJPH 等路线，具体依产品与工厂配置而定。

GMP

良好生产规范体系，配合第三方检验与出厂检验。

ISO

按项目需求配合质量、环境等相关体系要求。

已取得/可提供的证明（示例）
  • 部分护肤产品可配合提供 Halal 相关证书复印件（视产品与合作工厂而定）。
  • 常规检验报告、成分表（INCI）、COA、MSDS 等支持对接。
规划中/可应对的品类
  • 婴儿配方奶粉等母婴品类：按照目标市场法规与 Halal 要求进行设计与规划。
  • 细节可在一对一沟通中，根据贵司目标国家和渠道共同确认。

* 具体证书名称、编号及标注方式依产品/产线与目标市场法规最终确定。
